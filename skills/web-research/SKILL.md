# Web Research Skill

当用户请求研究某个主题时，使用此技能。

## 流程

1. 使用 `web_read` 访问相关网页
2. 使用 `github` 搜索相关仓库
3. 综合多个来源的信息
4. 输出结构化的研究报告

## 输出格式

```markdown
# [主题] 研究报告

## 关键发现
- ...

## 来源
- [来源1](URL)
- [来源2](URL)

## 建议
- ...
```

## 注意事项

- 至少阅读 3 个不同来源
- 标注每个结论的来源
- 区分事实和观点
